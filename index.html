    • MINHAS EXPERIÊNCIAS.

    HTML

    Adoro usar tags semânticas para dar sentido ao código. Sou fã de formulários modernos com validações HTML5 que fazem tudo funcionar sem esforço. Vídeos e áudios? Estou dentro, sempre com legendas para todos! Acessibilidade é meu mantra, seguindo o WCAG à risca. E claro, otimizo tudo com compressão de imagens e minificação de CSS e JS para deixar a página voando.

    CSS

    Transformo páginas com layouts flexíveis usando Flexbox e Grid. Adoro brincar com cores e gradientes, criando visuais de cair o queixo. Animações CSS? Tenho um truque ou dois na manga para dar vida aos elementos. Design responsivo é meu forte, garantindo que tudo fique incrível em qualquer dispositivo. E, claro, organizo meu CSS como ninguém, usando SASS para manter tudo limpinho e modular.

    JavaScript

    Estou começando a me aventurar no mundo do JavaScript! Já domino o básico, como manipulação do DOM para fazer a página responder ao usuário. Adoro usar eventos para dar interatividade, e já estou brincando com funções assíncronas para melhorar a experiência. Estou aprendendo sobre APIs para trazer dados dinâmicos, e começo a entender as maravilhas dos frameworks. Passo a passo, estou evoluindo e adorando cada descoberta!
